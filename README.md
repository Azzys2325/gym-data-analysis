## Project Overview
This project analyzes a dataset from "Iron Forge Gym" to identify patterns in member behavior, workout effectiveness, and peak hours.

## Key Technologies
- **Python 3.12**
- **Pandas** (Data Manipulation)
- **Matplotlib & Seaborn** (Data Visualization)

## Main Insights
- **Loyalty:** Monthly members show higher attendance than Annual members.
- **Top Workouts:** Strength Training and CrossFit lead in calories burned.
- **Operations:** Peak traffic identified at [TU HORA PICO].

## How to run
1. Clone the repo.
2. Install dependencies: `pip install pandas matplotlib seaborn`.
3. Open the `.ipynb` file in Jupyter or VS Code.
